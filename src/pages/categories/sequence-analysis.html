FastQC/MultiQC
Quality Control and Repor Aggregation

FastQC is a quality control tool for high-throughput sequencing data. I use it to assess the quality of raw sequence data before further analysis. MultiQC is a tool for aggregating results from multiple bioinformatics tools into a single report. I use it to summarize quality control metrics and analysis results.

Key Applications:

  • Quality assessment of raw sequencing reads
  • Detection of common sequencing issues
  • Generation of quality control reports
  • Visualization of read quality metrics
  • Aggregation of FastQC reports
# Run FastQC on a FASTQ file fastqc sample.fastq -o fastqc_reports # Generate HTML report fastqc sample.fastq -o fastqc_reports --extract # MultiQC to aggregate FastQC reports multiqc fastqc_reports -o multiqc_report # Generate HTML report multiqc . -o multiqc_report --force

HMMER
Profile Hidden Markov Models

HMMER is a tool for searching sequence databases for homologous sequences using profile hidden Markov models. I use it for protein family identification and functional annotation.

Key Applications:

  • Protein family identification
  • Functional annotation of sequences
  • Domain prediction
  • Phylogenetic analysis preparation
# HMMER search against a protein database hmmscan --tblout results.txt my_model.hmm protein_database.fasta # HMMER search with specific parameters hmmscan --cpu 4 --domtblout results.txt my_model.hmm protein_database.fasta
