Trimmomatic
Read Trimming

A flexible read trimming tool for Illumina NGS data. I use it to remove low-quality bases and adapter sequences from raw reads.

Key Applications:

  • Adapter removal
  • Quality filtering
  • Sliding window trimming
# Trimmomatic command example with sliding window and quality filtering # Cuts when average quality in a sliding window of 3 bases drops below 17 trimmomatic PE -phred33 WH_R1.fastq WH_R2.fastq trim/1.R1_clean.fastq trim/1.R1_unpaired.fastq trim/1.R2_clean.fastq trim/1.R2_unpaired.fastq SLIDINGWINDOW:3:17 LEADING:20 TRAILING:20 MINLEN:31 # Trimmomatic command example with maxinfo # Usefull for paired-end reads with uneven quality trimmomatic PE -phred33 WH_R1.fastq WH_R2.fastq trim/1.R1_clean.fastq trim/1.R1_unpaired.fastq trim/1.R2_clean.fastq trim/1.R2_unpaired.fastq MAXINFO:40:0.5 MINLEN:31

BWA (Burrows-Wheeler Aligner)
Alignment

BWA is a fast and accurate read aligner for short reads. I use it for mapping sequencing reads to reference genomes.

Key Applications:

  • Mapping short reads to reference genomes
  • Handling paired-end reads
  • Generating SAM/BAM files for downstream analysis
# BWA index the reference genome bwa index mycoplasma_ref.fasta # BWA mem for paired-end reads bwa mem mycoplasma_ref.fna ERR486840_1.fastq ERR486840_2.fastq > alineamiento.sam # BWA mem for single-end reads bwa mem mycoplasma_ref.fna ERR486840.fastq > alineamiento.sam

Samtools
Assembly

Samtools is a suite of programs for interacting with high-throughput sequencing data.

Key Applications:

  • Manipulating SAM/BAM files
  • Sorting, indexing, and viewing alignments
  • Variant calling and filtering
# Conversion and processing BAM files samtools view -S -b alineamiento.sam > alineamiento.bam # Sorting BAM files samtools sort alineamiento.bam -o alineamiento_sorted.bam # Indexing BAM files samtools index alineamiento_sorted.bam

Velvet
De Novo Assembly

Velvet is a tool for de novo assembly of short reads. I use it for assembling genomes from NGS data. Based on the de Bruijn graph algorithm.

Key Applications:

  • De novo assembly of short reads
  • Handling paired-end reads
  • Generating contigs and scaffolds
# Velvet assembly command # velveth: Builds the de Bruijn graph velveth velvet31 31 -shortPaired -fastq -separate ERR486840_1.fastq ERR486840_2.fastq # Velvetg: Assembles the graph into contigs velvetg velvet31 -exp_cov auto -cov_cutoff auto

MEGAHIT
Metagenomic Assembly

MEGAHIT is a fast and memory-efficient assembler for metagenomic sequencing data. Specially usefull for metagenomic NGS data. Based on conpressed Brujin graph (SdBG).

Key Applications:

  • De novo assembly of metagenomic data
  • Handling large and complex datasets
  • Generating contigs for downstream analysis
  • High efficient but can generate fragmented assemblies
# MEGAHIT assembly command for paired-end reads megahit -1 ERR486840_1.fastq -2 ERR486840_2.fastq -o megahit_output

Bowtie2
Alignment

Bowtie2 is a fast and memory-efficient aligner for short reads. I use it for mapping sequencing reads to reference genomes.

Key Applications:

  • Mapping short reads to reference genomes
  • Handling gapped alignments
  • Generating SAM/BAM files for downstream analysis
# Bowtie2 index the reference genome bowtie2-build mycoplasma_ref.fasta mycoplasma_ref # Bowtie2 for paired-end reads bowtie2 -x mycoplasma_ref -1 ERR486840_1.fastq -2 ERR486840_2.fastq -S alineamiento.sam # Bowtie2 for single-end reads bowtie2 -x mycoplasma_ref -U ERR486840.fastq -S alineamiento.sam
